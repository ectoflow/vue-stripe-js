# Vue Stripe.js

Full power of Stripe.js + beauty of Vue. Build advanced payment integrations quickly. Use examples for easy start with Payment, Card, and Express Checkout.

## Demo 

### 1. Clone
```bash
git clone git@github.com:ectoflow/vue-stripe-js.git
```

### 2. Install

```bash
npm install
```

### 3. Run demo
```bash
npm run dev
```
